# 图像下采样与混叠抑制实验

## 实验目的
理解图像下采样中的混叠现象，掌握抗混叠滤波的原理与方法，验证高斯滤波参数σ与下采样倍数M的理论关系，并实现基于梯度分析的自适应下采样。

## 实验环境
- Python 3.x
- OpenCV
- NumPy
- Matplotlib
- SciPy

## 实验原理

### 采样定理
根据奈奎斯特采样定理，当采样频率低于信号最高频率的两倍时，高频成分会混叠为低频伪影。图像下采样时，若不做预滤波，高频细节会产生摩尔纹等混叠现象。

### 抗混叠滤波
下采样前先进行低通滤波（如高斯滤波），滤除超过新奈奎斯特频率的高频成分，可有效抑制混叠。高斯滤波参数σ与下采样倍数M的理论关系为：σ = 0.45M。

### 自适应下采样
根据图像局部纹理复杂度动态调整σ，在纹理丰富区域用小σ保护细节，在平坦区域用大σ抑制混叠。

## 实验内容

### 第一部分：混叠分析
生成棋盘格测试图（256×256，块大小8）
生成Chirp测试图（水平方向频率从左到右递增）
对比直接下采样与高斯预滤波下采样的效果
通过FFT频谱分析混叠的频域表现

### 第二部分：验证σ公式
固定下采样倍数M=4
测试不同σ值：0.5、1.0、1.8、2.0、4.0
计算每种σ的MSE，找到最佳σ
与理论值σ=0.45M=1.8进行对比

### 第三部分：自适应下采样
使用梯度分析估计局部纹理复杂度
生成局部M值分布图（2~4范围）
计算局部σ = 0.45 × 局部M
对不同区域使用不同σ进行滤波和下采样
与统一σ方法对比误差图

## 实验结果

### 混叠现象
棋盘格直接下采样后出现明显的摩尔纹，Chirp信号右侧高频区域下采样后频率发生错误折叠。FFT频谱显示，直接下采样后高频能量向中心折叠，而预滤波后混叠消失。

### σ公式验证
M=4时，σ=0.5（太小）混叠残留明显，σ=4.0（太大）图像过度模糊，σ=1.8附近取得最低MSE，与理论值0.45×4=1.8吻合。

### 自适应下采样
自适应方法的整体MSE低于统一σ方法。误差图显示，自适应方法在高梯度区域（边缘）误差更小，在低梯度区域也能有效抑制混叠。

## 思考题
对于4倍下采样的人脸照片，人脸区域和背景区域不应使用相同的σ。人脸区域（五官、皮肤纹理）需要小σ（约1.0~1.4）保留细节；背景区域（墙壁、天空）需要大σ（约2.0~2.5）抑制混叠。可通过梯度估计、局部方差或边缘检测来决定各区域的σ值。

## 结论
预滤波是抑制下采样混叠的关键步骤，高斯滤波参数σ=0.45M是最优选择。自适应下采样根据图像内容动态调整参数，能够更好地平衡细节保留和混叠抑制，整体效果优于统一参数方法。
